Gestión de Intentos en Borrador

Resolución de problemas cuando un examen o test no se envía correctamente.

Cuando un intento de examen o actividad tipo test aparece en estado “Borrador”, significa que el estudiante no pulsó el botón Enviar, sino únicamente Guardar y cerrar.

Si el examen está configurado con envío automático al finalizar el tiempo, el sistema enviará el intento de forma automática una vez se agote el temporizador.

Esta opción se configura en los ajustes del examen. Puedes ver cómo activarla en la Guía de Ajustes de Exámenes.

En caso de que el envío automático no esté configurado, y el estudiante no haya pulsado Enviar, el intento no se registrará para su evaluación. La solución en este caso es:

  • Permitir que el estudiante acceda nuevamente al examen.
  • Indicarle que pulse el botón Enviar.
  • Configurar una excepción de examen para el alumno afectado, si fuera necesario.
Para ampliar el plazo o permitir un nuevo acceso individual, consulta la Guía de Excepciones.
